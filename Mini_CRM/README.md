# Mini CRM

A Customer Relationship Management (CRM) Web Application built using Flask and SQLite.

## Features

- Add Customer
- Edit Customer
- Delete Customer
- Search Customer
- Dashboard Statistics
- Responsive Design
- Dark Slate and Orange Theme

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
