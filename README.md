# Mini CRM

A Customer Relationship Management (CRM) Web Application built using Flask and SQLite.

## Features

- Add Customer
- Edit Customer
- Delete Customer
- Search Customer
- Export Customer Data to CSV
- Dashboard Statistics
- Responsive Design
- Modern UI with Purple and Navy Blue Theme

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Run Project

```bash
pip install -r requirements.txt
```

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
